# INTRODUCTION

## WHAT IS MAPC COMPONENTS?

MAPC (Metropolitan Area Planning Council) Components is a component library and starting guide for MAPC web projects run by the Digital Services team in service of the Greater Boston region of Massachusetts.
MAPC Components strives to make it straightforward for developers to apply consistent theming for MAPC web apps/sites by providing a set of thematic constants, configurations, and patterns for general usage, as well as a set of high-quality components that already make use of them. To be reused throughout the MAPC digital ecosystem to jumpstart the development process for each project.
