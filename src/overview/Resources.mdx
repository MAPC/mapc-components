## Accessibilty

### Layout

How we approach building sites that are [machine readable](https://www.smashingmagazine.com/2020/10/developing-semantic-web/) by assistive technologies.

Relevant `mapc-components` should include an reasonable default arial tag, free for customization as seen fit

### Colors and Contrast

`mapc-components` provides themes and styling with several MAPC branded color palettes. To check accessibilty:

- Check your UI with this [contrast checker](https://contrastchecker.com/).
- Check your UI for all types of [color blindness](https://accessgarage.wordpress.com/2013/02/09/458/).
