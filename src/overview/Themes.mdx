# Themes

These are several sets of MAPC branded styling themes, including guidelines for: Styles, Colors, Fonts

## Styling

### Global Styled Components

Import the global style theming from MAPC components to the file

```
import MapcGlobalStyle from "mapc_components";
```

Add this code block to the top of your React tree to inject MAPC global styles to your project:

```
<MapcGlobalStyle />

```
